# v0.1.2

## Features

- **Adaptive timeout response:** The parallelization factor is now halved immediately when a worker reports a task timeout, rather than waiting for the next evaluation window. This lets overloaded workers recover faster by shedding load aggressively. The factor floors at 1 and exits steady mode so the controller can re-probe upward once conditions improve.

## Fixes

- **Docker build with hatch-vcs:** Fixed `uv sync` failing inside Docker due to missing `README.md` and absent git history. The Dockerfile now copies `README.md` and sets `SETUPTOOLS_SCM_PRETEND_VERSION` as a fallback for hatch-vcs.
